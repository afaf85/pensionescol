# pensionescol

Colombian pension analysis tool for Colombians abroad, expats in Canada, and workers evaluating retirement options.

## Purpose
This project helps analyze pension scenarios in Colombia, including contribution history, retirement options, and planning paths for people living outside Colombia.

## Planned features
- pension calculator
- contribution gap analysis
- retirement eligibility checks
- guidance for expats in Canada
- comparison of pension options
